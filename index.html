Values

Values represent a single number. They can be combined to form an expression. As we can't overload operators in JS, we use method calls such as .subtract(27). Values objects track their relationship to other Values in an expression to form an expression graph.

Gradients

You can compute the gradient of an expression by calling .backward() on the result of an expression. This gradient will help us modify the weights of a neural network when training it.

Neuron

We can use Values to build a Neuron for our neural network. A neuron takes a set of weighted inputs, applies an activation function, and spits out an output. The gradient of the weights tells us how modifying them will change the output.
